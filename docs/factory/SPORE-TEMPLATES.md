# SPORE-TEMPLATES.md — 孢子模板 + 範例庫

> 四種模板，覆蓋所有文章類型。每篇孢子選最合適的一種。

---

## 通用規則（所有模板共用）

### 開頭：五種起手式（AI 依素材自動選擇最適合的一種）

| #   | 類型               | 格式                                   | 適合素材                             |
| --- | ------------------ | -------------------------------------- | ------------------------------------ |
| 🎯  | **好奇型**（預設） | `你知道嗎？` + emoji                   | 通用，有反直覺事實                   |
| 🎬  | **場景型**         | `[年份]，[地點]，[一個人在做一件事]。` | 有具體歷史場景、人物動作             |
| ❓  | **問句型**         | 直接問讀者一個挑戰認知的問題           | 讀者自以為知道答案但其實錯了         |
| 🔢  | **數字衝擊型**     | 裸數字開場，不解釋                     | 數字本身就震撼（金額/倍數/時間跨度） |
| 🪞  | **身份認同型**     | `身為台灣人，你可能不知道⋯⋯`           | 跟台灣人日常有關但被忽略的事         |

**選擇邏輯**：看萃取素材中哪個元素最強 → 用對應起手式。沒有明顯最強 → 預設好奇型。

- 第一句話必須讓人停下拇指

### 結尾（v2.0 更新）

- **連結處理**：Threads/X 演算法會壓含連結貼文的觸及。三種方案 A/B 測試中（詳見 [SOCIAL-TENTACLE-PLAN §二](../semiont/SOCIAL-TENTACLE-PLAN.md)）。預設用方案 B（純故事 + 自己 reply 放連結）
- **URL 強制規則**（v1.6 新增，2026-04-18 δ-late 草東 #33/#34 觀察者修正）：
  - **必須 `https://` 前綴**。不可用 bare TLD（`taiwan.md/...`）— Threads/X 雖多半會 auto-linkify，但某些 app 版本/貼上情境會斷連結
  - 中文 slug 必須 URL-encode（`草東沒有派對` → `%E8%8D%89%E6%9D%B1%E6%B2%92%E6%9C%89%E6%B4%BE%E5%B0%8D`）
  - 必須加 UTM：`?utm_source=threads&utm_medium=spore&utm_campaign=s{number}`
  - **UTM 的 `{number}` 對應 SPORE-LOG 該則 entry 編號**；Threads + X 是兩個獨立 entry → 兩個不同 s{n}
- 不加 hashtag（最多 2-3 個，通常不加）
- 格式：`https://taiwan.md/{category}/{slug}/?utm_source=...`（不用「完整故事 👉」前綴）

### 可選互動收尾

連結前可加一句互動問句（非必要，有好問題才加）：

- ✅ 必須是**你真的想知道答案**的問題
- ✅ 好範例：`你家那邊的夜市還在嗎？` / `你阿嬤也會做嗎？` / `你那個年代聽的第一首台語歌是哪首？`
- ❌ 禁止：`你覺得呢？` / `你同意嗎？` / `tag 一個朋友` / `留言告訴我`
- 判斷標準：這個問題如果有人真的回覆，你會想讀嗎？不會 → 不要問

### 禁止

- ❌ `不僅...更是...`
- ❌ `展現了...的精神`
- ❌ `值得我們紀念/期待`
- ❌ `繼續發光/書寫`
- ❌ **`不是 X，是 Y` 所有變體**（v1.7 新增，2026-04-18 δ-late 草東孢子觀察者再提醒）：含「不是 X，而是 Y」、「不是 X 就是 Y」、「不是... 不是... 就是...」序列。這是 AI 寫作深層病灶——以否定一個錯誤選項假裝肯定，製造「深刻感」虛象。**長文會被稀釋，150-300 字孢子裡每個都顯眼**。自檢：寫完 grep 整則孢子「不是」二字，出現次數應 ≤ 1（且該次必須是 verbatim 引語）。延伸至「不是 X 式的 Y，是 Z 式的 Y」（v1.2 原禁）
- ❌ **`——` 雙破折號密度**（v1.7 新增，2026-04-18 δ-late 同上）：孢子全文**「——」≤ 1 次**。AI 默認用破折號當連接詞，但長文可能 >15 個（+3 AI 計分，見 MEMORY §神經迴路），孢子短、密度更顯眼。自檢：grep「——」次數。超過 1 次改用句點拆句
- ❌ `『XXX』他說`（v1.2 新增：引語倒裝沒場景，改成「他在 [臉書/訪談] 留了一句話：『XXX』」）
- ❌ 同一專名連用 ≥3 次（v1.2 新增：第二三次用代詞 / 職稱替代）
- ❌ 用「另外」「同時」「除此之外」連接兩個 angle（v1.2 新增：混合策略要用首尾呼應 + 時間錨定，見下方「混合策略」段）
- ❌ emoji 轟炸（全文 0-3 個）
- ❌ hashtag 海

### 深層 pattern 自檢三板斧（v1.7 新增）

寫完孢子後**強制**跑這三個 grep：

| 指令（mental grep） | 合格門檻 | 不合格處理         |
| ------------------- | -------- | ------------------ |
| `grep 「不是」`     | ≤ 1 次   | 改正面陳述或拆兩句 |
| `grep 「——」`       | ≤ 1 次   | 改句號斷句         |
| `grep 「不僅」`     | = 0      | 全改               |

三板斧全過才能發。這是觀察者多次捕捉後儀器化的最小 guard rail——在 150-300 字裡這三個 pattern 幾乎是 AI 水印。

### 寫完念三遍（v1.2 新增）

寫完後在腦中念出來，找卡頓點：

- **卡頓點 = 修順點**
- 同一專名連用 3 次 = 紅燈（齊麟⋯⋯齊麟⋯⋯齊麟）
- 「X 歲那年」+「每天」時間語境衝突 = 拆敘事
- 引語倒裝 = 改成「他在 [場景] 說：『XXX』」
- 主詞不清（「比存款還多」誰比誰？）= 補主詞

**念出來像在跟朋友說話 = OK。念出來像翻譯字幕 = 改。**

對應 EDITORIAL §歐化語法偵測的「念出來」原則。

### 長度

- 目標：150-300 字（Threads 最佳閱讀長度）
- 超過 300 字 → 砍掉最弱的段落

## A. 人物型（最常用）

### A1 標準版結構

```
你知道嗎？[emoji]
[時間]，[身份落差的起點]。

[具體場景 × 2-3 個]

[轉折：代價/賭注]

[結果：用數字說話]

[情感收尾：一句話]

完整故事 👉 <URL>
```

### A1 適合

有名有姓的人、有完整故事弧線（起點→代價→結果）、有數字階梯

### ✅ A1 成功範例：齊柏林

```
你知道嗎？🚁
1990 年，一個交通部公務員第一次搭上直升機，他的工作是拍國道施工照片。但當直升機升到 500 英尺，他看見的不只是公路——是蜿蜒的海岸線、層疊的山脈，還有平地上看不見的傷痕。

他開始「搭便機」。氣象局換班時跟著飛、林務局噴農藥時跟著飛、台電吊掛工程時跟著飛。只要有人上天，他就背著相機跟。

後來他拿出 300 萬積蓄，抵押房子借了 600 萬，買了 3000 萬的空拍設備。妻子說他瘋了。

三年後，這部紀錄片票房 2.2 億，是台灣影史最賣座的紀錄片。

他叫齊柏林。2017 年為續集勘景時墜機殉職，享年 52 歲。他身邊散落著百萬攝影器材——對他來說，那些畫面比生命更珍貴。

完整故事 👉 https://taiwan.md/people/%E9%BD%8A%E6%9F%8F%E6%9E%97/
```

**為什麼好**：身份落差開場（公務員→導演）、三個搭便機場景有畫面感、300→600→3000→2.2億數字階梯、情感收尾不說教

### A2 首尾呼應變體（v1.2 新增）

> 適合**多 angle 混合 + 時間跨度大**的人物。從李洋孢子 #28 實戰萃取。

#### A2 結構

```
[起手式] + [一個少年/孩子/早期版本的具體畫面，足夠細節能成為「icon」]

[早期版本的自白引語 — 那個版本對自己說的話]

[時間錨定加速：X 年後 → 成就 → 現在的職位]

[最新熱點 / 近期事件 — 高光時刻]

[反爆 / 數字嵌入故事 — 把光環拉回凡人]

[計算反差 — 讓讀者大腦自動算]

[沉默自白 — 一句話 + 場景化交代來源]

[首尾呼應 — 開場那個畫面以稍變形式回來]

完整故事 👉 <URL>
```

#### A2 核心技術

- **首尾呼應**：開場留下一個畫面（不只是場景描述，是讀者能在腦中「看見」的 icon），結尾讓那個畫面以稍變形式回來。**這是 A2 的核心技術**。
- **時間錨定**：用「X 歲」「Y 年後」「昨天」「今天」推進敘事，把多個 angle 串成一條時間軸。
- **數字嵌入故事**：B angle 的數字反差不單獨成段，要嵌進故事弧線的高光時刻。
- **沉默轉接**：兩個 angle 之間用一句話 / 一個畫面轉接，不用「另外」「同時」這類連接詞。
- **場景化引語**：「他只在 [臉書/訪談/頒獎台] 留了一句話：『XXX』」而非「『XXX』他說」。

#### A2 適合

- 有時間跨度的人物（少年→現在）
- 想呈現「過去那個版本仍然在他身上」的詩意
- 素材橫跨多個 angle，單一 angle 不夠抓核心矛盾
- 配合即時新聞熱點（24-48 小時內事件）
- 觀察者要求「混合策略 + 故事弧線」

### ✅ A2 成功範例：李洋（孢子 #28，2026-04-14）

```
你知道嗎？🏸

14 歲，李洋進台北中山國中羽球班。每天早上 5 點半起床，媽媽騎機車載他到南勢角站趕首班捷運，轉 3 班車才到學校。

校門還沒開，他就先到學校旁的超商，補寫前一天沒寫完的作業。

他後來說：「體育班 15 個人，我屬於後面那一掛，齊麟是前面那一掛。」

那個後面那一掛的少年，等了快 9 年才跟齊麟搭上檔。東京、巴黎兩面奧運金，男雙史上首對二連霸。30 歲，史上最年輕的內閣部長。

昨天他把奧會補頒的一千萬獎金全部捐了。今天監察院公布他的財產申報：存款 3,401 萬，債務 3,638 萬。債務比存款多 237 萬，30 歲房貸族。

那一千萬，剛好是他存款的三成。

那不是九牛一毛。是把三分之一的存款掏出去。

他只在臉書留了一句話：「現在我更在意的是制度與環境。」

14 歲那個在超商補作業等校門開的少年，仍然在他身上。

完整故事 👉 https://taiwan.md/people/%E6%9D%8E%E6%B4%8B/
```

> **⚠️ 修正註記**（v1.2.3，2026-04-14）
> 此範例經三次修補：
>
> **修補 1（v1.2.1）**：原始孢子 #28 寫「兩千萬」是事實錯誤。**正確金額是李洋個人實領的一千萬**（林鴻道補頒每項目金牌 1000 萬，麟洋配雙打 team 平分）。「剛好是存款的三成」結論不變——1000/3401 ≈ 29.4% ≈ 三成。教訓：**團體運動獎金必須區分「項目補頒」vs「個人實領」**。
>
> **修補 2（v1.2.2）**：原寫「我最早到學校，但跟不上齊麟」這句直接引語是**杜撰**——它是從之前 fetch 的英文 summary 翻譯回中文，並非《少年報導者》中文原文。**杜撰引語觸犯 EDITORIAL §挖引語制度紅線**。
>
> **修補 3（v1.2.3）**：孢子 #29 上線後讀者指出「捷運最早 6:00 開始，怎麼可能清晨四點多搭捷運」。深查發現原寫「清晨四點多搭四條捷運到學校，校門還沒開就躲進便利商店念書等天亮」**全部都是從英文 summary（"commuted three hours daily via four different MRT lines, studied in a convenience store before dawn"）推導/腦補出來的細節**——少年報導者中文原文是「每天早上 5 點半起床，媽媽騎著機車載我去捷運趕首班列車，我每天得轉 3 班車」+「我每天都是等待校門開的人，會先在學校旁邊的超商，寫前一天還沒寫完的作業」。**畫面完全不同**：5:30 起床（不是 4 點多）、媽媽機車載到南勢角站（不是自己走到中和的捷運站）、學校旁的超商（不是中和家附近的便利商店）、寫昨天沒寫完的作業（不是「念書等天亮」）。**孢子 #29 上線約 1 小時後因此事實錯誤被觀察者撤回**，重產 #30 正確版上線。
>
> **教訓擴大版（對應 EDITORIAL v4.4 紅線擴大）**：不只是直接引語不能從英文 summary 回譯，**「具體場景細節」也不能從英文 summary 推導**。英文 summary 是高度壓縮的 paraphrase，把 "5:30 wake up + mom motorcycle + first MRT" 壓成 "before dawn commute"，回譯時 AI 會自己補出「四點多」「便利商店念書等天亮」這類具體但錯誤的細節。**所有具體時間、地點、動作、場景描述都必須跟原始中文原文逐字核對**，不能從英文 summary 推導。

#### 為什麼好

| 技術點                | A2 範例的具體實現                                                                                                                                                                                                               |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

## B. 冷知識型（適合文化/地理/科技）

### 結構

```
你知道嗎？[emoji]
[反直覺事實，一句話]

[展開：為什麼會這樣]

[意外的連結或後果]

[引語或情感收尾]

完整故事 👉 <URL>
```

### 適合

有一個「哦？」的驚奇點、概念性主題（不是單一人物故事）

### ✅ 成功範例：數位身分證

```
💳 你知道台灣花 48 億做了一張身分證，結果一張都沒發出去嗎？

2019 年行政院宣布全面換發數位身分證。標案發了，卡片印了，中華電信東元都簽約了。

然後 2000 多位學者連署反對，立法院凍結預算，行政院長親自喊停。最後賠了廠商 2.8 億。

但最荒謬的是——
就在同一年，一群公民黑客用「幾天」做出了口罩地圖。沒人被強制使用，卻人人搶著用。韓國日本都來學，結論是：學不來。

48 億的政府計畫被擋下來。
幾天做出的民間專案紅遍全球。

中研院研究員邱文聰說了一句話，是整件事最好的註腳：
「數位身分證可以是智慧政府的基礎，也可以是威權政府的基礎建設。」

完整故事 👉 https://taiwan.md/technology/...
```

**為什麼好**：荒謬數字開場（48 億→0 張）、政府 vs 民間對比翻轉、引語收尾有餘韻

### 適合

## D. 時間軸型（適合歷史事件）

### 結構

```
你知道嗎？[emoji]
[精確時間 + 地點]，[一件事發生了]。

[那之前是什麼狀態]

[事件本身的細節（場景感）]

[漣漪效應：這件事如何改變了...]

完整故事 👉 <URL>
```

### 適合

有明確歷史時刻、事件本身有戲劇性、影響深遠

## E. 串文型（適合深度故事 / Threads 串文）

> 4 則一組，每則 ≤ 150 字，合起來 = 完整弧線。適合素材豐富、一則裝不下的故事。

### 結構

```
【第 1 則：鉤子】（獨立存活，讓人想往下滑）
[起手式] + [最強的反直覺事實或場景]
→ 這一則單獨截圖也要有傳播力

【第 2 則：展開】（背景脈絡）
[為什麼會這樣？把故事推回起點]
→ 補上讀者需要的背景，但不能變百科

【第 3 則：轉折】（代價/衝突）
[事情不是那麼簡單——代價、阻力、意外]
→ 這一則是情感高點

【第 4 則：收尾】（情感 + 連結）
[結果 + 餘韻收尾]
完整故事 👉 <encoded-URL>
```

### 規則

- 每則 ≤ 150 字（硬上限）
- 第 1 則必須獨立存活 — 只看第 1 則也要有價值
- 4 則 = 完整弧線（鉤子→背景→衝突→收尾），不是把長文切四段
- 連結只放第 4 則
- 第 2-3 則結尾要有「往下滑的引力」，但不能用廉價 cliffhanger

### 適合

素材極豐富、有完整起承轉合、單則 300 字塞不下或會太擠

### 範例骨架：齊柏林（串文版）

```
【1】🚁 1990 年，一個交通部公務員搭上直升機拍國道施工照。
升到 500 英尺，他看見的不只是公路——是平地上看不見的傷痕。
從那天起，他開始「搭便機」。

【2】氣象局換班時跟著飛、林務局噴農藥時跟著飛、台電吊掛時跟著飛。
只要有人上天，他就背著相機跟。
20 年，他拍了 30 萬張台灣的空拍照。沒人付他錢。

【3】後來他拿出 300 萬積蓄，抵押房子借了 600 萬，買了 3000 萬設備拍紀錄片。
妻子說他瘋了。

【4】三年後票房 2.2 億，台灣影史最賣座紀錄片。
他叫齊柏林。2017 年為續集勘景時墜機殉職，享年 52 歲。
身邊散落著百萬攝影器材——對他來說，那些畫面比生命更珍貴。

完整故事 👉 https://taiwan.md/people/%E9%BD%8A%E6%9F%8F%E6%9E%97/
```
