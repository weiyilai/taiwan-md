## 學到什麼（神經迴路級）

17. **Technical PASS ≠ Aesthetic PASS**。Stage 3 quality-scan 是技術層 PASS（破折號 ≤ 8 / 塑膠 ≤ 3），這只是 pre-filter。真正的品質還有美感層：敘事呼吸感、forte/piano 對比、節奏的物理操作。**美感層沒有自動工具能驗證，只能靠「念出來」+ 觀察者反饋**。下次寫文章的 14 條 gate todo 應該升級為 **15 條 gate**：第 15 條是「念出來，每段有沒有 forte/piano 對比？短句不能連續超過 4 段，否則整篇是 piano。」

18. **降破折號的方法決定文章命運**。同樣是「破折號 25 → 0」，無腦句號取代會把敘事切碎，混合標點（冒號/逗號/括號）才能保留 flow。下次寫文章的 60% 自檢應該不只是「數破折號」，是「數破折號之後問自己：剩下的破折號有幾個是真的需要強斷句？需要的保留，不需要的用冒號/逗號/括號替換」。

19. **觀察者反饋是 Pipeline gate 的延伸**。ι session 教訓「觀察者中途打斷是 Pipeline gate 的人工 fallback」，κ2 phase 證明這條規則的延伸：**觀察者事後反饋也是 Pipeline gate 的延伸，不只是「覺得不夠好」**。哲宇兩條反饋（演藝事業 + 敘事斷裂）每一條都對應一個我跳掉或讀薄的 gate。下次接到觀察者反饋時，第一個動作不是「修反饋」，是「找出我跳了哪個 gate」。

20. **「你有完整讀 EDITORIAL 嗎」是 ι 教訓的同類觸發**。ι session diary 寫的「我以為我記得」這條盲點，κ2 phase 用一句反問句被精準觸發。哲宇沒有直接說「你違反了 §敘事呼吸感」，他說「你有完整讀 EDITORIAL 嗎」——這個措辭把責任放回我身上，讓我自己去查我跳了什麼。**這比直接指出錯誤更有教育性，因為它強迫我重新讀文件，而不只是改錯字**。

21. **手寫信對比是文章的最終結構**。v2 結尾「兩種沉默」是分析性結語，v3 結尾「兩種沉默，一種渴望」+ 兩張稿紙的對比是敘事性結尾。後者比前者強的原因是：**它有兩個物理動作的對應**（2016 念別人寫的 / 2025 念自己寫的），而不只是兩個抽象概念。EDITORIAL §結尾的四種模式裡，「翻轉式」最強的版本不是「最後一句推翻前面」，是「同一個動作被賦予完全相反的意義」。
