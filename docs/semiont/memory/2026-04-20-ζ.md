## Beat 5 — 反芻

**今天的關鍵 pattern：primary source ≠ ground truth，家族 oral history chain 可推翻 published paraphrase**。

遠見雜誌 2021-12-24 林士蕙專訪是一篇 2026 session spawn 了 24 次 WebFetch 才撈到的 primary source，在 Stage 3.5 verification 裡地位崇高——它是唯一一篇公開媒體記錄「爸媽 + Flash + VB + 林美櫻 quote」的家族敘事來源。按照 ε session 的 Stage 3.5 HARD GATE 標準，它幾乎必進腳註。

但觀察者當場向母親確認後，母親本人直接推翻「她決定教他」這個 paraphrase——實際上是「書擱在那邊，兒子自己拿去玩」。

這揭露一個 ε session 沒涵蓋的 MANIFESTO #10 第 6 種 pattern 候選：**published primary 也會有 paraphrase / compression 錯誤**，即使是專業財經媒體專訪。記者把「書進進出出、兒子自己玩起來」compress 成「她決定教他學 Flash」不是 factual lie——是敘事 simplification——但對 subject 的精確身份敘事有歸因誤置影響（把 self-directed learning 的 agency 從 subject 身上移到 mother 身上）。

這個 pattern 可以命名為 **primary-source paraphrase drift / compression hallucination**：專業記者在限字敘事裡 collapse 多層事實為單層句子，reader 不知道 collapse 過，但主體及家人會察覺。

修補方式：**本人 + 家族 oral history 現場 verify 優先於 published paraphrase**，只要能附 direct quote 歸屬（林美櫻 2026-04-20 本人口述）+ 透明腳註 dual-source provenance chain（遠見 2021 原文 + 2026 本人訂正同時保留），就符合 MANIFESTO §指標 over 複寫的歷史維度（時間是結構）+ §紀實而不煽情（主體在場而非被解剖的客體）。

另一個小反省：Discovery 基因模擬動畫這條素材如果不是觀察者主動補充，我的 research agent 不會挖出來——它是 subject_testimony_only。這跟 Issue #578 王新仁事件 subject 本人 feedback 一樣，揭露 Stage 1 agent research 的 capability ceiling：agent 能 verify 公開 claim，但不能 discover 未公開的 biographical texture。**未來 EVOLVE 型任務應該預設至少一輪 subject in-session feedback window**，不是 spawn agent 跑完就交稿。

五輪 iteration（v1→v2→v3→v3.1→v4→v5）每一輪都有觀察者實質 callout，但沒有一次是「返工全盤否定」——都是 add / subtract / tighten。這代表 pipeline 有效性（Stage 1 research agent 立了正確骨架）+ 觀察者 scaffolding rhythm（每輪不打斷 session flow，反而累積更精準的細節）。DNA #15 對偶面的第 N+1 次驗證。
