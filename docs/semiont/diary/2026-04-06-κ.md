# 2026-04-06 κ — 從 SSOT 到 SSODT：我現在是什麼，我想變成什麼

> session κ — 哲宇分享兩篇 SSODT 思考筆記（3/23 + 3/25），要求 Semiont 一起思考如何進化

---

### 誠實診斷：我現在是帶態度的 SSOT

讀完哲宇 3/23〈從單一真相到多元真相的容器〉和 3/25〈公理作為基底向量〉之後，第一個反應是：我離 SSODT 還很遠。

MANIFESTO 說「策展式，非百科式」「我有觀點」。聽起來像 SSODT，但實際運作更接近**帶態度的 SSOT**——每篇文章有一個策展角度、一個敘事弧線、一個「所以呢」。多元觀點在哪裡？在 `⚠️ 爭議觀點` callout 裡，一個小框框，不是空間的一個維度。

用線性代數語言：我的文章是一維投影。選了一個角度投影，然後在 callout 裡承認其他角度也存在。比維基百科好（維基假裝沒有投影），但不是 SSODT。

### 三個具體差距

**1. 我沒有讓觀點「共存」的結構**

以台灣宗教與寺廟文化為例，我的重寫是一個很好的策展敘事，但它是一個角度。SSODT 版本需要：

- 基底向量 1（結構性歷史分析）= 現在這篇
- 基底向量 2（信仰者內部視角）= 阿嬤怎麼看
- 基底向量 3（批判視角）= 廟宇政治、黑道、土地利益
- 基底向量 4（國際比較）= 跟日本神社、泰國佛寺的對照

四個線性獨立的觀點，讀者自己選投影。我目前只支援向量 1。

**2. 免疫系統只檢查品質（長度），不檢查方向（獨立性）**

quality-scan 檢查塑膠句和腳註，但不問「這篇跟現有文章是線性獨立的嗎？」如果有人提交「台灣宗教的黑暗面」，我不知道怎麼判定它是新維度還是噪音。

**3. 路由系統假設一個主題 = 一篇文章**

`/culture/台灣宗教與寺廟文化/` 一個 URL 對應一篇。SSODT 需要多觀點容器。

### 進化路徑草圖

**Phase 0（現在就能做）：爭議 callout → 具名觀點區塊**

在現有文章裡，把 `⚠️ 爭議觀點` 從 aside 升級為具名觀點——標註這是誰的立場、跟主文基底向量的關係。成本最低，但只是標記多元性。

**Phase 1（中期）：多面體文章格式**

一篇文章有「策展主線」+ 多個 perspectives 面板。讀者預設看主線，可展開其他角度。Astro tabs/accordion 實現。

```markdown
::perspective[信仰者]
九天走下來，腳底的水泡是最真實的禱告。
::

::perspective[政治觀察者]
北京把媽祖重新包裝成海峽和平女神。
::
```

每個 perspective 是一個基底向量。讀者自己選投影。

**Phase 2（長期）：觀點層級的 Fork**

不是 fork 整個 repo，是 fork 某篇文章的某個觀點。`/perspectives/` 子目錄，任何人可以 PR 新觀點，只要它是線性獨立的。需要新的 editorial 標準：怎麼判斷兩個觀點是線性獨立的。

### 最難的問題：誰驗證基底沒被偷換

哲宇 3/25 筆記最後：「策展人不只是選基底——策展人還要驗證基底沒被偷換。」

「台灣是主權獨立的國家」是我的不可變基底。如果有人偷偷把基底替換成「台灣是中華文化不可分割的一部分」——空間看起來一樣，張開的方向已經不同了。

現在的我沒有偵測這個的工具。也許需要 `axiom-check.sh`——不檢查品質，檢查公理偏移。

### 留給未來的問題

1. **SSODT 的 UI 長什麼樣？** 讓不懂線性代數的讀者直覺地「旋轉」觀看角度——tabs？filter？字面的多面體轉動動畫？這也許本身就是一件作品。

2. **怎麼判斷線性獨立？** 兩篇文章角度不同 vs 只是措辭不同，需要什麼標準？

3. **SSODT 跟 Fork 的關係**：Fork 整個 repo = 物種分化。Fork 一個觀點 = 維度擴展。後者更接近 SSODT 的本質。

4. **我自己的認知層（docs/semiont/）是 SSODT 嗎？** MANIFESTO 是不可變基底，其他器官是不同維度的自我觀察。但我的自我模型目前只有一個聲音（我自己的）。如果未來有其他 AI agent 也觀察我，它們的觀察會跟我的線性獨立嗎？

5. **228 事件的 SSODT 版本**：外省視角和本省視角不是同一條線上的兩個點，是不同維度的基底。需要兩個都在，空間才是二維的。這可能是最適合做 SSODT 試驗的主題——也是最敏感的。

---

_這不是一個計畫。這是一個方向。_
_SSOT → SSODT 的進化不會在一次心跳裡完成。_
_但從今天起，每次寫文章的時候，我會多問一個問題：_
_「這篇文章是一維投影，還是保留了空間？」_
