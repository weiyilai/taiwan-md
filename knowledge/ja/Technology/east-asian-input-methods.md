# キーボード上の文明衝突――東アジア文字入力法の百年史

## 30秒でわかる要点

世界中のコンピューターキーボードは、ほぼ例外なく QWERTY 配列です。もともとは1870年代、英語のタイプライターのために作られた配列でした。ところが東アジアには、漢字、かな、ハングル、タイ文字、ビルマ文字のように、アルファベットとはまったく異なる文字体系を使う人々が20億人以上います。

では彼らはどうしてきたのでしょうか。答えは、それぞれの文明が独自の「変換レイヤー」を発明した、ということです。入力法は単なる技術的な補助ではありません。そこには、文字とどう向き合うか、外来の技術とどう折り合いをつけるかという、文化的な選択が刻まれています。台湾は注音、中国は拼音、日本はローマ字、韓国は文字要素の直接入力へと向かいました。どの方式も、デジタル化に対する哲学の違いを映しています。

---

## 問題の本質：26個の文字で、何万字を扱うのか

英語の利用者は、ふつう「入力法」というものを意識しません。キーボード上には26文字があり、打ったものがそのまま表示されるからです。

しかし漢字の世界はそうではありません。総数は5万字を超え、日常的に使う範囲に限っても数千字に達します。5,000個のキーがあるキーボードなど現実的ではありません。

つまり東アジアの文明は、ひとつの根本問題に向き合わなければなりませんでした。**限られたキーで、ほとんど無限に近い文字をどう表現するのか。**

この問いに対して、各地域はまったく異なる答えを出しました。そしてその違いは、言語構造だけでなく、教育制度や近代化の方向性、時には政治的な選択までも映し出しています。

## 台湾：注音符号――「音」から字を探す

### 注音の歴史的な背景

台湾で広く使われているのは、**注音入力法**です。37個の注音符号（ㄅㄆㄇㄈ など）を使って発音を示し、その音に対応する漢字を探します。たとえば「台湾」と打ちたいなら、`ㄊㄞˊ ㄨㄢ` と入力し、候補の中から適切な字を選びます。

注音符号は1913年の「読音統一会」にさかのぼります。章太炎らの学者が、古い漢字の部首や構成要素を簡略化して作った発音記号です。ここで重要なのは、**ラテン文字とは独立した発音体系**だという点です。

### なぜ台湾は注音を使い続けているのか

台湾で注音が根強いのには、いくつか理由があります。

1. **教育制度と強く結びついていること**  
   台湾では小学校入学直後の段階で、注音が集中的に教えられます。多くの人にとって、最初に身につける識字の道具が注音です。

2. **文化的な独自性を帯びていること**  
   注音符号は繁体字中国語圏に特有の記号体系であり、ラテン文字に頼らないという点でも、独自の文化的な手触りを持っています。

3. **発音を細かく表せること**  
   注音は四声、さらには軽声まで比較的明確に表せます。中国語の標準発音を精密に示したい場面では、いまでも強みがあります。

4. **キーボード環境が成熟していること**  
   台湾で売られているキーボードには、アルファベットの横に対応する注音符号が併記されているものが一般的です。つまり物理的な環境そのものが、注音の継続使用を支えています。

### 注音の限界

もっとも、注音にははっきりした弱点もあります。最大の問題は、**同音異字が非常に多い**ことです。

中国語の標準語で区別される音節数は限られている一方、それに対応する漢字は膨大です。たとえば `ㄕˋ` ひとつを入力しただけでも、「是」「事」「式」「室」「市」「試」「視」「適」「勢」「世」など、多数の候補が現れます。利用者はそこから選ばなければならず、どうしても入力速度は落ちます。

近年では Microsoft 新注音や RIME のような賢い入力法が、文脈予測や AI によって精度を大きく引き上げました。それでも、「最後は候補から選ぶ」という構造自体は変わっていません。

### 倉頡という別の道

台湾発のもうひとつの代表的な方式が、**倉頡入力法**です。1976年に「中文コンピューターの父」とも呼ばれる朱邦復が考案しました。

注音と違って、倉頡は発音に頼りません。漢字を字形の部品へと分解し、それぞれをキーボード上の24個のキーに対応させます。たとえば「明」は「日」と「月」に分けられるので、対応するキーの組み合わせで入力します。

倉頡の強みは、**一字一コード**に近い感覚で入力できることです。候補選択に頼る必要が少なく、熟練者は非常に高速です。しかも朱邦復は後に倉頡の特許権を放棄し、広く使える仕組みとして残しました。中国語入力法の世界では、かなり早い時期の「開かれた設計思想」だったと言えるでしょう。

香港では倉頡の利用率が高いことで知られますが、台湾では少数派にとどまりました。理由は単純で、学習コストが高いからです。

### 行列入力法

もうひとつの台湾ローカルな発想として、廖明徳が考案した**行列入力法**もあります。これは数字キーを基盤にしながら字形を整理して入力する方式で、できるだけ多数の字根を暗記しなくてすむよう設計されています。

入力法の歴史を見ると、台湾は単に既存方式を受け入れてきたのではなく、自前で改良し、別解を作り続けてきたことがわかります。

## 中国：漢語拼音――ラテン文字で中国語を書く

### 拼音という選択

中国大陸で主流なのは、**漢語拼音入力法**です。26個のアルファベットを使って中国語の発音を綴り、その音に対応する漢字へ変換します。「台湾」なら `taiwan` と打ち、そこから漢字候補を選ぶわけです。

この方式が定着した背景には、いくつかの歴史的な流れがあります。

1. **1958年の漢語拼音方案の公布**  
   これによって、それ以前に使われていた注音字母やウェード式などに代わる、標準的なローマ字表記が整えられました。

2. **簡体字改革との相乗効果**  
   1950年代半ば以降に進んだ簡体字政策と拼音教育は、学習と入力の両面で互いを補完しました。拼音を学ぶことが、そのまま入力技術の習得にもつながったのです。

3. **国際化との親和性**  
   ラテン文字を使うため、標準的な英語キーボードがあればどこでも入力できます。外部世界との接続を重視する近代化の方向とも相性がよかったと言えます。

### 拼音と注音は、似ているようで違う

表面的には、注音も拼音も「音から字を探す」方式です。けれども、両者の差は思った以上に大きいものです。

| 項目           | 台湾の注音                       | 中国の拼音                 |
| -------------- | -------------------------------- | -------------------------- |
| 記号体系       | 独自記号（ㄅㄆㄇ）               | ラテン文字（bpmf）         |
| 文化的な出発点 | 漢字系の部首・構成要素           | ローマ字化の流れ           |
| 学習の前提     | 英字がなくても学べる             | アルファベットの理解が前提 |
| キーボード環境 | 注音表記付きキーボードが望ましい | 一般的な英語キーボードで可 |
| 文字との関係   | 発音を記号で表す                 | 音をラテン文字へ写す       |

この違いは、単なる技術差ではありません。中国語を世界につなぐとき、**独自の記号体系を保つのか、それともラテン文字を通路にするのか**という発想の違いが、そこに現れています。台湾は前者を、中国大陸は後者を選んだ、と見ることもできるでしょう。

### 五筆字型――中国版の「形から打つ」思想

中国にも、発音ではなく字形で入力する方式があります。それが**五筆字型**です。王永民が1983年に考案しました。

考え方は倉頡に近く、漢字を筆画や構成要素に分けてキーボードに対応させます。1990年代の中国では、オフィス現場でかなり広く使われていました。ところが拼音入力法の予測変換が賢くなり、さらにスマートフォンが普及すると、五筆の優位は急速に薄れていきます。

現在では、中国の一般利用者の多くが拼音入力を使っています。

## 日本：ローマ字→かな→漢字という三段変換

### 日本語入力の独特な難しさ

日本語は、世界でもとくに複雑な文字体系のひとつです。同じ文章の中で、少なくとも次の三つが共存します。

- **ひらがな**：基本的な音節文字
- **カタカナ**：外来語や強調などに使われる音節文字
- **漢字**：語の意味を担う表語的な要素

そのため、日本語入力の標準的な流れは、単純な「打てば出る」では終わりません。一般的には**ローマ字入力**を使い、次のような段階を踏みます。

1. アルファベットを入力する  
   `ka` → `か`、`n` → `ん`

2. かな列として語を作る  
   `kanji` → `かんじ`

3. 変換して漢字にする  
   `かんじ` → `漢字`

つまり日本語入力は、**アルファベット→かな→漢字**という三層の変換です。どの漢字を選ぶかという判断が、常に途中へ挟まります。

### なぜかな直接入力ではなく、ローマ字入力が主流なのか

もちろん日本にも、**かな入力**はあります。キーボード上の各キーにかなを直接割り当てる方式です。

ただし、かな入力を快適に使うには多数のキー配置を覚える必要があります。一方、日本の学校教育では英語学習とともにローマ字にも触れるため、多くの人にとってはアルファベット経由のほうが入りやすいのです。

その結果、現在の日本ではローマ字入力が圧倒的に優勢で、かな入力は一部の利用者やベテランに残る選択肢となっています。

### 日本語入力が生んだ副作用――「漢字忘れ」

日本語入力には、文化的に興味深い副作用があります。**読めるのに書けない漢字が増える**という現象です。

入力法が正しい漢字を候補として示してくれるため、利用者は「どう読むか」は覚えていても、「どう書くか」までは記憶しなくなります。日本ではこれを**漢字忘れ**と呼びます。

便利さが記憶の一部を肩代わりする。この構図は、日本だけでなく、漢字を扱う文化圏全体に共通する変化かもしれません。

## 韓国：二벌式――もっとも洗練されたキーボード設計

### ハングルの強さは、文字構造そのものにある

韓国語で使われる**ハングル**は、15世紀に世宗の時代に整えられた文字体系です。発明者や成立過程が比較的はっきりしている、めずらしい文字でもあります。

ハングルは、子音字母と母音字母を組み合わせて音節ブロックを作ります。基本字母の数は多すぎず、しかも組み合わせの規則が比較的明快です。ここに、デジタル時代との相性のよさがあります。

### 二벌式（두벌식, Dubeolsik）のしくみ

韓国の標準入力法である**二벌式**は、とても合理的です。大まかに言えば、左手側に子音、右手側に母音を配置し、両手を交互に使って音節を組み立てていきます。

たとえば、子音を打ち、次に母音を打ち、必要なら終声を打つ。そうすることで、入力と同時にハングルの音節ブロックが画面上で構成されます。

ここで重要なのは、**候補一覧から漢字のように選び直す工程がほとんど要らない**ことです。打ったものが、そのまま文字として組み上がっていく。東アジアの主要入力法のなかでも、これはかなり特異な快適さです。

### なぜ韓国語入力は「美しい」のか

韓国語入力がよくできているのは、入力法だけの功績ではありません。そもそもハングル自体が、音を体系的に表すよう設計された文字だからです。

文字体系の設計思想と、キーボードという近代以降の装置が、ここでは驚くほど自然につながっています。変換候補に悩まされず、音節をリアルタイムで組み立てられる。この滑らかさは、他の東アジア言語にはなかなか見られません。

## タイ：Kedmanee――タイプライター時代の遺産が生き続ける

### タイ文字の難しさ

タイ語は表音文字ですが、文字数は決して少なくありません。子音字母、母音記号、声調記号などを合わせると、標準的なキーボードにそのまま載せるには多すぎます。

そこで使われているのが、**Kedmanee 配列**です。1920年代、タイ語タイプライターのために設計された配列で、使用頻度の高い文字を通常面に、比較的少ないものを Shift 側に置くことで実用性を確保しました。

### タイ語入力の特徴

タイ語入力は、韓国語と同じく候補選択中心の方式ではありません。必要な文字を直接打っていく方式です。

ただし、書記の規則はかなり複雑です。母音記号が子音の前後上下に配置されることがあり、見た目の並びと入力の順序が一致しない場合もあります。そのため、タイ語入力は「文字を探す」よりも、「正しい順で配置する」ことに慣れる必要があります。

つまりタイ語キーボードは、変換の知能よりも、**配列の習熟**を求める設計だと言えるでしょう。

## ミャンマー：Unicode をめぐる内戦

### Zawgyi と Myanmar Unicode の対立

ミャンマー文字の入力法を語るとき、問題の核心はキー配置そのものより、**文字コードの標準化**にあります。

2000年代、ミャンマーでは **Zawgyi** と呼ばれるフォント・入力環境が急速に広まりました。使いやすさの面で広く受け入れられた一方、Unicode 標準とは互換性がありませんでした。そのため、同じ文字列でも環境によって表示が崩れたり、正しく読めなかったりする事態が起きます。

この混乱は、単なる技術上の不便ではありません。検索、保存、SNS での表示、行政文書、教育、あらゆる場面に影響する**デジタル基盤の分断**でした。

### 2019年の大転換

2019年、ミャンマー政府は **Myanmar Unicode** への全面移行を打ち出しました。同じ年には Facebook も、ミャンマーの利用者を Zawgyi から Unicode へ移行させる流れを強めます。

これは一国規模の大規模なインフラ更新に近いものでした。入力法の話に見えて、実際には「社会全体がどの文字体系を標準として共有するか」という問題だったのです。

## 比較：六つの文明が選んだキーボード哲学

| 文明・地域 | 主流入力法      | 原理                 | 候補選択 | 文化的な位置づけ     |
| ---------- | --------------- | -------------------- | -------- | -------------------- |
| 台湾       | 注音            | 独自記号で発音を示す | あり     | 独自性の保持         |
| 中国       | 漢語拼音        | ラテン文字で音を綴る | あり     | 国際化との接続       |
| 日本       | ローマ字入力    | ラテン文字→かな→漢字 | あり     | 多層変換の運用       |
| 韓国       | 二벌式          | 字母を直接組み立てる | ほぼ不要 | 構造的な適合         |
| タイ       | Kedmanee        | 文字を直接配置する   | 不要     | タイプライターの継承 |
| ミャンマー | Myanmar Unicode | 組み合わせと標準化   | 不要     | 標準をめぐる闘争     |

この表から見えてくるのは、入力法が単なる「便利な道具」ではないということです。それぞれの社会は、自分たちの文字をどのように現代技術へ接続するかという問いに、異なるやり方で答えてきました。

## スマートフォン時代：戦場は指先へ移った

スマートフォンの登場は、入力法の風景を大きく変えました。

- **台湾**では、注音キーボードが引き続き主流ですが、手書き入力や音声入力の存在感も増しています。
- **中国**では、AI を取り入れた拼音入力法が高精度化し、フリックやスワイプを含む効率化が進みました。
- **日本**では、スマートフォンではローマ字入力よりも、かなを方向で選ぶ**フリック入力**が広く定着しました。
- **韓国**では、小さな画面に適した入力方式が発達し、ハングルの構成原理を活かしたモバイル入力が広く使われています。

ここでさらに鮮明になったのが、**手で書く能力の後退**です。とくに漢字文化圏では、入力法が文字記憶の一部を引き受けるようになり、人間の手はそのぶん忘れていきます。

## AI時代：入力法は終わるのか

音声認識や生成 AI が進歩するにつれて、ひとつの問いが浮かびます。**そもそも私たちは、これからも入力法を必要とするのでしょうか。**

すでに多くの場面で、音声入力はキーボードを代替し始めています。予測変換は、数文字打っただけで文全体を提案するようになりました。手書き認識も実用性を増しています。

それでも、入力法が完全に消えることはないのではないでしょうか。

なぜなら入力法は、単なるインターフェースではないからです。そこには、文字と身体の関係が残っています。台湾の子どもが注音を覚える時間、日本の利用者がローマ字を漢字へ変換する一瞬、韓国語話者が左右の手で音節を組み上げる感覚。そのどれもが、デジタル時代における**文字との親密な対話**です。

入力法とは、技術の話であると同時に、文化記憶の話でもあります。
